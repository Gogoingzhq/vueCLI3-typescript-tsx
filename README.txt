1、Vue CLI 需要 Node.js 8.9 或更高版本 (推荐 8.11.0+)  检查node版本
2、npm install -g @vue/cli     vue --version 安装cli3
3、vue create vuecli3-ts  创建项目-回车
4、 default (babel, eslint)
      Manually select features 选择手动选择功能
5、Unit Testing和E2E Testing可以不选择，其他选中
安装时，可以参考：https://www.jb51.net/article/138703.htm
6、默认还是采用.vue格式  可以改为tsx，然后采用render函数加载模板
